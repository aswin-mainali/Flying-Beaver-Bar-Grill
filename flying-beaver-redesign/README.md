# The Flying Beaver Website Redesign

A fully responsive single-page restaurant website built with HTML, CSS, and vanilla JavaScript.

## How to use
1. Open `index.html` in a browser.
2. To deploy on Netlify, drag and drop this whole folder or upload the ZIP.
3. Replace images in `/assets` if the restaurant wants newer photos.

## Functionality included
- Sticky desktop navigation
- Mobile hamburger menu
- Responsive layout
- Real restaurant image assets from the current website
- Menu, call, directions, and social links
- Automatic “Today” highlight in Daily Features
- Gallery lightbox
- Mobile bottom action bar
